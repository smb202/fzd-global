Migrating 200+ dashboards from Power BI to Grafana.

A Tier-1 European insurer needed to consolidate two reporting platforms and reduce licensing cost while preserving every business-critical KPI.

Challenge

  • 200+ dashboards on Power BI
  • Migration complexity, adoption risk
  • Inconsistent KPI definitions
  • Slow refresh on legacy semantic models

What we built

  • Migration framework & mapping tool
  • Dashboard redesign for Grafana
  • Performance & refresh logic
  • Governance and ownership model

Technologies

Power BI · Grafana · SQL Server · Azure · Python migration tooling

Business impact

  • Modernized reporting estate
  • Improved dashboard performance

  • Reduced platform licensing cost

Business impact
